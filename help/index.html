Elements

Sand, Metal, Water, Lava, Cobblestone, Obsidian, Fire, Plant, Wood, Acid, Ice, Oil, Gunpowder, Snow, Bomb, Steam, Smoke, Ash and Eraser are defined in the simulator. Each will receive a dedicated subsection with behavior, density, reactions and implementation details.

Physics Engine

The simulator updates powders, liquids, gases and solids independently using specialized update functions. Density, gravity, combustion, weather, explosions and particle effects are processed each frame.
